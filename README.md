# 🌸 Art of Perfume — Quiz de Perfumes

Quiz interactivo que descubre tu perfil olfativo y te recomienda perfumes según tu personalidad.

## 🚀 Publicar en GitHub Pages

### Paso 1: Crear repositorio en GitHub
1. Ve a [github.com/new](https://github.com/new)
2. Nombre del repositorio: `art-of-perfume` (o el que prefieras)
3. Marca **Public**
4. Click en **Create repository**

### Paso 2: Subir archivos
Sube toda la carpeta del proyecto a tu repositorio. Asegúrate de que contenga:

```
📁 tu-repositorio/
├── index.html          ← Página principal (obligatorio)
├── README.md           ← Este archivo
└── 📁 images/
    ├── logo.jpg        ← Tu logotipo de Art of Perfume
    ├── art-of-perfume-dupe.jpeg  ← Imagen del producto dupe
    ├── art-of-perfume-preview.jpeg  ← (Opcional) Preview para redes sociales
    └── favicon.png     ← (Opcional) Icono de la pestaña del navegador
```

### Paso 3: Agregar tus imágenes
Coloca tus imágenes en la carpeta `images/`:
- **`logo.jpg`** → Tu logotipo de Art of Perfume (el archivo original `logotipo art of perfume.JPG`)
- **`art-of-perfume-dupe.jpeg`** → La imagen del producto (el archivo original `art of perfume gemini.jpeg`)

### Paso 4: Activar GitHub Pages
1. En tu repositorio ve a **Settings** → **Pages**
2. En **Source** selecciona **Deploy from a branch**
3. En **Branch** selecciona `main` y carpeta `/ (root)`
4. Click en **Save**
5. Espera 1-2 minutos y tu página estará disponible en:
   ```
   https://TU-USUARIO.github.io/art-of-perfume/
   ```

## 📱 Características
- ✅ Diseño responsive (móvil, tablet, escritorio)
- ✅ Quiz de 8 preguntas con 4 perfiles olfativos
- ✅ Recomendaciones personalizadas por género

## ⚠️ Notas importantes
- El archivo **debe** llamarse `index.html` (GitHub Pages lo usa como página principal)
- Las imágenes de los perfumes se cargan desde URLs externas (fimgs.net) y no necesitan ser descargadas
- Si el logo o la imagen del dupe no cargan, se muestran placeholders automáticamente
